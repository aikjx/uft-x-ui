## 💡 快速修改

### 修改主题颜色
编辑 `tailwind.config.js`:
```javascript
colors: {
  primary: {
    500: '#your-color',  // 修改主色调
  }
}
```

### 添加新公式
编辑 `src/data/formulas.ts`:
```typescript
{
  id: 18,
  name: '新公式名称',
  latex: '\\vec{F} = m\\vec{a}',
  // ...
}
```

### 修改页面内容
编辑对应的 `.vue` 文件：
- `src/views/HomeView.vue` - 首页
- `src/views/FormulasView.vue` - 公式列表
- `src/views/FormulaDetailView.vue` - 公式详情
